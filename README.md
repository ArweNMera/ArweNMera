## 📌 Proyectos destacados

### 🎓 OptiAcademic

Sistema académico para la gestión y generación de horarios mediante un motor de restricciones CSP.

**Características principales:**

- Gestión de usuarios por roles.
- Administración de docentes, estudiantes, cursos, aulas y secciones.
- Generación de horarios institucionales.
- Generación de horarios para estudiantes.
- Backend con autenticación JWT.
- Base de datos relacional en MySQL.

**Tecnologías:**  
`FastAPI` · `React` · `MySQL` · `SQLAlchemy` · `JWT` · `Vite`

---

### 📦 Sistema de Inventario

Sistema orientado al control de productos, entradas, salidas y stock disponible.

**Características principales:**

- Registro de productos.
- Control de entradas y salidas.
- Cálculo automático de stock.
- Organización simple para almacén.
- Diseño fácil de entender y actualizar.

**Tecnologías:**  
`Excel` · `Bases de datos` · `Gestión de inventario`
